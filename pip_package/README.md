# Todoer

Save todos from anywhere!

## Instructions

1. Install:

```
pip install todoer
```

2. Generate an aesthetic ASCII visual:

```python
from todoer import todoer
todoer = todoer.Todoer()
todoer.save_todo('lfg')
```

3. Ride the space skyway home to 80s Miami
